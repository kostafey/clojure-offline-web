# Пример простого блога на Clojure

Пример задеплоен здесь: http://helloworld-writing.rhcloud.com/articles

Никакого логина не требуется. Реализованы операции CRUD для объекта article.

## Running

Запускать приложение так:

    lein ring server

War-ку собирать так:
    
    lein ring uberwar
